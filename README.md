# Market Place API.

## Technology Stack.
 - SpringBoot.
 - Hibernate ORM.
 - H2 database.
 - HikariCP.
 - Jersey.
 - JSON.
 - Lombok.
 - Logback.

## Prerequisites.
 - JDK 8.
 - Maven.

## How to Run.

``` mvn clean install```

```mvn package  && java -jar target/market-place-0.1.0.jar```

## H2 console Access.
```http://localhost:8080/h2-console```
